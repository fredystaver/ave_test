# Тестовое задание для AVE TECHNOLOGIES
Сервис апускается в корне проекта командой `docker-compose up --build`
Swagger доступен по адресу: http://0.0.0.0:8888/docs#

Для проверки правильной работы приложения нужно попасть в интерфейс редиса:
1. Выполняем команду`docker compose ps`
2. Копируем имя контейнера с редисом `ave_test-redis-1`
3. Выполняем команду `docker exec -it ave_test-redis-1 sh` (или bash)
4. Входим в интерфейс редиса путем использования команды `redis-cli -a redis12345`
5. Проверяем получение/создание/обновление/удаление данных после отправки запроса в API
